# ProjetoAurora
🚀 Sistema de Triagem Operacional - Missão Aurora SigerEste projeto consiste em um algoritmo de automação em Python desenvolvido para
realizar a análise de telemetria e eficiência energética de foguetes orbitais comerciais. O sistema atua como uma **Triagem Assistida por IA**, classificando
a viabilidade de lançamentos com base em parâmetros técnicos reais.

  # 📋 Funcionalidades
  - Análise energética: Monitoramento de capacidade total (kWh), consumo estimado e perdas energéticas;
  - Algoritmo de decisão: Classificação automática entre "MISSÃO EM RISCO" ou "MISSÃO ESTÁVEL";
  - Identificação de anomalias: Sugestão de riscos baseadas em desvios de padrões de segurança (ex: perdas > 6%).

  # 📊 Dados de Referência
  - Falcon 9 (SpaceX): Foco em integridade operacional e redundância;
  - New Glenn (Blue Origin): Análise de pressurização autogéna e alta capacidade de carga;
  - Electron (Rocket Lab): Monitoramento de bombas elétricas e baterias de alta perfomance;
  - Terran T (Relativity Space): Avaliação de tecnologias de impressão 3D em estágio de desenvolvimento.

  # 🛠️ Tecnologias Utilizadas
  - Python 3.14.3
  - Jupyter Notebook (.ipynb)
  - VS Code IDE

  # 🚀 Como Executar
  1. Certifique-se de ter Python e a extensão Jupyter Notebook instalado no VsCode;
  2. Abra o arquivo script_python.ipynb;
  3. Selecione Kernel Pythpn no canto superior direito (preferencialmente .venv);
  4. Execute as células em ordem (Shift + Enter) para visualizar o relatório de triagem.

  # 🌎 Reflexão Ética e Sustentabilidade
   O projeto integra uma análise sobre o impacto social e ambiental da exploração espacial, destacando: 
  - A transição para combustíveis menos poluentes como o Metanol (LNG);
  - A importância da reutilização de componentes para a economia circular; 
  - A responsabilidade ética na validação de sistemas atuomatizados para evitar falhas catastróficas.
